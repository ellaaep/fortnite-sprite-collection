# Fortnite Sprite Collection

A private fan-made Fortnite Sprite collection tracker.

## What it does

- Saves your collected Sprites in the browser with `localStorage`.
- Missing Sprites are dark / grayscale.
- Collected Sprites are full color with a check mark.
- Variant progress bars use matching variant colors.
- Click a variant progress card, like Gold, to filter that variant.
- Click any Sprite to open details.
- Export / Import your progress as JSON before moving to another URL or browser.

## Open locally

Open `index.html` in your browser.

Or run:

```bash
python3 -m http.server 5173
```

Then open:

```text
http://localhost:5173
```

## Deploy

Use Vercel as a static project. Keep the repository private if you want.

Build command: empty  
Output directory: `.`
